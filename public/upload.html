How it works

  1. Your ZIP should contain one folder per product, e.g. AL-1026 Graphite Black/.
  2. Each folder holds the product's images (PIC 1.png, PIC 2.jpeg…).
  3. The first image (natural order) becomes the cover; the rest form the gallery.
  4. Model number & finish are detected from the folder name automatically.
  5. Thumbnails and optimised web images are generated on import.
  6. A flat ZIP of loose images becomes a single product named after the ZIP.

Tip: re-uploading the same folder creates a new product entry. Use the gallery to manage or remove duplicates.
